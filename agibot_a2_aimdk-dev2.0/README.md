
# 智元远征 A2 - AimDK

本仓库用于 **智元远征 A2** 机器人的软件二次开发，包含协议、文档、示例、预构建协议文件等。

## 1. 协议

开发者可调用的所有接口对应的原始协议都存放在 [protocol](./protocol) 路径下，按照协议类型分为 [ros2](./protocol/ros2/) 和 [protobuf](./protocol/protobuf/) 两个大目录，每个大目录下再按功能板块进行划分。

## 2. 文档

您可以直接参考 [智元远征 A2 官方文档页面](https://open.agibot.com/docs/aimdk) 来了解机器人的详细使用、开发方式。

## 3. 示例

开发者可以参考本仓库中的示例，来加快上手熟悉的速度。

示例代码存放在 [examples](./examples/) 路径下，针对于所有的接口都提供了示例。

## 4. 预构建协议文件

开发中需要用到的协议文件已经预构建好，存放在 [prebuilt](./prebuilt/) 路径下，请根据文档的提示选择合适的协议文件。
